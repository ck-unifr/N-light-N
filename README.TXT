The content of this folder is organized as follows:

N-light-N.pdf
    This is the documentation of the framework.

N-light-N.jar
    This is the compiled version of the framework. You
    can use it and include it as library in your Java
    applications.

LICENSE

src
    This folder contains the source files of N-light-N.

lib

    N-light-N.
